# Job Listing Portal 🚀

A full-stack web application designed to connect **recruiters** and **job seekers**. Recruiters can post job opportunities, and job seekers can browse and search jobs easily based on different criteria.

---

## 🎯 Purpose of the Project

This project is built to simulate a real-world job portal system where:

- 👨‍💼 **Recruiters** can post job openings
- 👩‍💻 **Job seekers** can explore available jobs
- 🔍 Users can search jobs using title, description, skills, or location

---

## ✨ Key Features

### For Recruiters
- Add new job postings
- Provide job details like title, description, experience, skills, and location

### For Job Seekers
- View all available job listings
- Search jobs by:
  - Job title
  - Description
  - Skills
  - Location
